Chrome 61 支持 [script(type='module')](http://caniuse.com/#feat=es6-module) 了！
* defer & async
* demo

## Defer & Async
* defer 是渲染完执行
* async 是下载后执行

![asyncdefer](https://html.spec.whatwg.org/images/asyncdefer.svg)


## Demo

html
```html
<script type='module'>
    import { sayHello } from './utils.js'
    sayHello();
</script>
```

utils.js
```js
export const sayHello = () => {
    console.log('hello');
    alert('hello');
};
```
