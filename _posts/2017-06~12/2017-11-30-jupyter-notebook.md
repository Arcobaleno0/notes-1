## Install jupyter
* Download `get-pip.py` from [https://bootstrap.pypa.io/get-pip.py](https://bootstrap.pypa.io/get-pip.py)
* `sudo python get-pip.py`
* `pip install --user jupyter`
* `cd /Users/xxx/Library/Python/2.7/bin`
* `./jupyter notebook`
