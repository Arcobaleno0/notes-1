一些工具的配置老是忘记，每次用到都要搜索，所以，写个 list 配置一下
* npm

### npm
> 设置 npm 淘宝镜像加速
```
// 设置淘宝镜像
npm config set registry https://registry.npm.taobao.org
// 查看配置
npm config list
// 恢复配置，以便正常在 npm 登录，发布包等操作
npm config delete registry
```
